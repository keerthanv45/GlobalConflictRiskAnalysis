# Global Conflict Intelligence Dashboard

A Big Data Analytics and Machine Learning project designed to analyze and visualize global conflict risk using economic, governance, and conflict-related datasets.

## Features

- Interactive Streamlit Dashboard
- Global Conflict Risk Visualization
- World Map Analysis
- KPI-Based Insights
- Conflict Trend Analysis
- Risk Category Classification (Low, Medium, High)
- Machine Learning using Random Forest
- Dynamic Filtering by Country and Year

## Datasets Used

- World Bank GDP Dataset
- World Bank Inflation Dataset
- World Bank Unemployment Dataset
- V-Dem Governance Dataset
- UCDP Georeferenced Event Dataset (GED)

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit
- Jupyter Notebook

## Project Objective

To integrate economic, governance, and conflict-related indicators into a unified analytical platform capable of identifying conflict patterns, assessing geopolitical risk, and providing data-driven insights through machine learning and interactive visualizations.

## Outcome

Successfully developed a conflict intelligence dashboard that classifies countries into risk categories and enables users to explore global conflict trends through interactive maps, charts, and analytics.
